This plugin allows you to automatically integrate CartRescuer tracking scripts into your Wordpress site.

== Description ==

What does CartRescuer do?
CartRescuer Recovers lost customers into sales. CartRescuer offers three services in one offering: Cart Abandonment, Surveys and Popups.

Why do you need it? 
70%-96% of visitors abandon before making a purchase. 15% of the top e-retailers are not doing any form of recovery or conversion optimization. Our Cart Abandonment service stops your sales funnel leakage.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `cartrescuer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register a CartRescuer account at http://cartrescuer.com/plan/
4. Once registered, copy your Tracking Code on the Implementation page (https://cartrescuer.com/implementation/)
5. Back in your Wordpress Admin, go to the CartRescuer Plugin Settings Page, paste the Tracking Code and Save Settings.
